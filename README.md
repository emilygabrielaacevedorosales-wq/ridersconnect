## 📱 ¿Qué es RidersConnect?

RidersConnect es una **red social y herramienta colaborativa** diseñada 
exclusivamente para conductores de plataformas de transporte como 
Uber, DiDi, Cabify, entre otras.

Los conductores pueden:

- 🗳️ **Votar tarifas justas** para cada ruta de su ciudad
- 💬 **Chatear en tiempo real** con otros conductores
- 🗺️ **Ver zonas calientes** con mayor demanda
- 📱 **Publicar y compartir** experiencias como red social
- 🏆 **Ganar badges** por participar activamente

---

## 🚀 Funcionalidades

| Función | Descripción |
|---------|------------|
| 🏠 Feed Social | Publica, comenta y da like a posts de otros conductores |
| 💰 Tarifas Justas | Propón y vota el precio óptimo para cada ruta |
| 🗺️ Zonas Calientes | Mapa con zonas de alta demanda y surge pricing |
| 💬 Chat Grupal | Salas de chat por tema: general, tarifas, emergencias |
| 👤 Perfil | Estadísticas personales, badges y logros |
